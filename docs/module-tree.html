Module Hierarchy

  • ABYSS_launch: This program is used to assemble all fasta files in a directory using the ABYSS tool. The assembly will be done with different lengths of kmère (20, 30, 40, 50, 60, 70, 80 and 90). A new pipeline has been created, please watch the Assembly_pipeline in the same folder
  • Alignement: This program is used to align multiple RNA-seq on a genome and merge the different alignments into a file
  • BRAKER_pipeline: This program is used to annotate with BRAKER. This script use :
  • Correction_Transcript: This program is used to correct the orthofinder result. They remove alternatif transcript of a same Orthologue group.
  • HmmerPipeline: This program is used to search a HMM profil for search EffecteurMAX. This program used HMMbuild et HMMsearch.
  • MODULES_SEB: Use it to import very handy functions.
  • OG_pylho: This script was created for a special case (phylogenetic OG). So they have no documentation. Please contact the author for more information
  • Quality: This program is used to retrieve quality data of all the assemblies done by the script ABYSS_launch
  • QualityAssemblage: This program is used to check assembly quality
  • Recup_T0_Only: This program is used to remove all alternative transcript of protein fasta file.
  • ToggleJobArray: This program launches the tool toogle in job array to be able to launch little by little the job
  • Verif_Annotation: This program is used to verif annotation (start or stop error or multi-stop)
  • comparaisonSecretome: This program is used to retrieve and compare information from the output of secretome prediction tools (signalP, targetP and Phobius). This program is used by secretome_Pipeline
  • correction_orthofinder: This program is used to correct the result of Orthofinder. They use a tblastn for search the OG sequence in the blast DB of all genome. Please, this script is working slowly and it's not finish, choose the correction_pipeline.
  • elimateREmotif: This program is used to eliminate the result of PS-scan with PS00014 pattern (RE target Motif). This program is used by secretome_Pipeline.
  • forRNAseq_inPlanta: This program is used to align multiple RNA-seq on a host genome for eliminate inPlanta RNA
  • formatFastaName: This Programme filter sequences by length
  • module_Flo: Use it to import very handy functions.
  • psiblast_tools: This program is used to align sequence with psiblast tools and give in supplementary a fasta file.
  • recupSeqFromGFF: This program is used to recup Seq from GFF format
  • renameAndAddParent
  • renameGFF: This Programme rename genes name in GFF file. The name of every gene will be : Mo_+NumGene+T+Numtranscript
  • repeatMasker_build: This program is used to create a bash file that can launch jobs on the cluster that uses repeatMasker on all the fasta files in a directory.
  • script_pangenome: This program is used to construct a pan génome. This script create a core genome File and a accessory genome file which contain only the group of orthologue of Effecteur MAX.
  • secretome_Pipeline: This program is used to predict secretome with SignalP, TagetP, Phobius.
  • selection_TMHMM: This program is used to selected the proteins having no transmembrane domain or only one in the 40 first amino acid. The program take in input the output of TMHMM Tools. This program is used by secretome_Pipeline.
  • stat_repeatMasker: This program is used to retrieve the statistique of result from repeatMasker_build.py script
