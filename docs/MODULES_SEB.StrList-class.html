Class StrList

source code

Classe qui dérive de list et qui vérifie que la variable ajouter dans l'objet (list) est bien dans la liste "expectedValue"

Example:

>>> x=StrList(expectedValue=['true'])
>>> x.append("true")
>>> x
['true']
>>> x.append("false")
