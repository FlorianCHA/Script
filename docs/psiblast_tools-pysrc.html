 19          Example

 22          >>> psiblast_tools.py -d /work/database/databaseName -f /work/database/query.fasta -o /work/result.fasta -i 10 -opt alignement -n exemple

 24          Help Programm

 28                  - \-h, --help

 32                                                   
 33          Input  infos for running: 
 34                  - \-f <path/to/fasta/file>, --fasta <path/to/fasta/file>
